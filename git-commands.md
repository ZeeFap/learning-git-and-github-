# Git Commands Cheat Sheet

## Basics
- `git init` — init repo
- `git status` — check changes
- `git add .` — stage all
- `git commit -m "message"` — commit
- `git log --oneline` — history

## Branches
- `git branch` — list
- `git checkout -b feature` — new branch
- `git merge feature` — merge
